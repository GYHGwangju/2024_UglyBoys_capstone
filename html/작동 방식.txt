# 파일 구성
index.html = 초기 화면 
page1.html = 카테고리, 카테고리에 따른 메뉴와 선택한 주문을 보여줌. 주문 목록 아래엔 주문하기 버튼이 있음
page2.html = 메뉴를 선택하면 나오는 옵션 페이지

순서
1. index 페이지에서 '시작하기' 버튼을 누르면 page1 로 넘어와서 메뉴를 고른다.
2. page1에서 각각의 메뉴를 보여주고, 카테고리를 선택하면 같은 화면 안에서 메뉴의 사진과 메뉴의 설명을 바꿔야 한다.
3. page1에서 메뉴를 선택하면 page2로 넘어가서 옵션을 고른다.
4. page2에서 옵션을 고른 다음에 확정 버튼을 누르면 page1에서 고른 메뉴와 page2에서 선택한 옵션을 저장하여 page1으로 돌아가 주문목록 아래의 주문내용에 업데이트한다.
5. page1에서 주문 버튼을 누르면 주문을 한다.
